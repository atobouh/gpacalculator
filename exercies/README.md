# Exercises (Dart)

This directory contains the three requested Dart exercises in separate files.

## Files

- `dart/exercise1_model_zoo.dart`
  - Abstract class, inheritance, and polymorphism
- `dart/exercise2_network_state_sealed.dart`
  - Sealed class states and exhaustive pattern matching
- `dart/exercise3_drawable_shapes.dart`
  - Interface-based design with drawable implementations
- `dart/run_all_exercises.dart`
  - Runs all exercises in one command

## Run

```powershell
dart exercies/dart/run_all_exercises.dart
```
